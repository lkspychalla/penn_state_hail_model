# penn_state_hail_model
This model simulates the trajectories, growth, and melting of hailstones through a three-dimensional storm simulation. Initially built by Matt Kumjian and Kelly Lombardo (Kumjian and Lombardo 2020), updated for non-spherical hailstones by Yuzhu Lin (Lin et al. 2024), with physics modifications by Lydia Spychalla (Spychalla et al. in prep).
